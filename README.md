# Project Euler em Português

Este repositório contém traduções para português dos problemas do Project Euler, com o objetivo de facilitar o estudo de matemática, lógica e programação para falantes da língua portuguesa.

## Sobre

O Project Euler é uma coleção de problemas matemáticos e computacionais que exigem não apenas conhecimento matemático, mas também a criação de algoritmos eficientes.

Site oficial: https://projecteuler.net

## Objetivo

Este projeto tem fins exclusivamente educacionais e não comerciais.

O objetivo é disponibilizar versões em português dos problemas para auxiliar estudantes interessados em matemática, algoritmos e programação competitiva.
